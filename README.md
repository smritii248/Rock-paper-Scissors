
# Rock, Paper, Scissors Game

An engaging Rock, Paper, Scissors game built using Python and Pygame.  
Features smooth animations, 2-player mode, a 20-second timer per round, score tracking, and a leaderboard saving the top 5 scores.

## Features

- Animated rock, paper, and scissors moves  
- Play against a friend in 2-player mode  
- 20-second timer per round to keep gameplay fast and exciting  
- Score tracking for both players  
- Leaderboard that saves and displays the top 5 high scores
